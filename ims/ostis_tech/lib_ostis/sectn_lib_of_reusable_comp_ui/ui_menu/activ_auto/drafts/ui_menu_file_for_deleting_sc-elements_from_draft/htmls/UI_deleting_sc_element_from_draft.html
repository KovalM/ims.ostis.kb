Команда пользовательского интерфейса удаления sc-элемента из черновика предназначена для удаления sc-элемента из черновика. Имеет два аргумента: знак удаляемого sc-элемента и знак черновика, из которого нудно удалить sc-элемент. Результатом выполнения команды является удалие указаного sc-элемента из указанного черновика.
