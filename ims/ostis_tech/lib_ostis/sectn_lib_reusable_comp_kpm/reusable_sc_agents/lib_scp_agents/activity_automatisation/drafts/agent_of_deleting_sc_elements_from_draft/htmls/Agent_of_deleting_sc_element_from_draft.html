Задачей агента удаления sc-элемента из черновика является удаление sc-элемента из черновика. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу удаления sc-элемента из черновика. Имеет два аргумента: знак удаляемого sc-элемента и знак черновика, из которого нудно удалить sc-элемент.
